# Trisulfide

A simple, zero-dependency TikTok user info scraper.

## Installation

```bash
npm install trisulfide
```
Usage

1. As a CLI Tool
```bash
npx trisulfide
```

3. In Your Code
```javascript
const trisulfide = require('trisulfide');

trisulfide('khabylame').then(data => {
  console.log(data);
});
```
